# Data-Leverager-Power-BI-ETL-Power-Query-Transformation-Project
This project focuses on building an end-to-end ETL pipeline using Power BI Power Query, including data extraction from web and Excel sources, data cleaning, transformation, and integration. It demonstrates key techniques like appending, merging, pivoting, and data profiling.
